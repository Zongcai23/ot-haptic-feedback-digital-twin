# Vision Module Slot

This folder is reserved for the **pose and depth estimation** module described in the paper.

Suggested future contents:
- `Pose_and_Depth.ipynb` or `pose_depth.ipynb`
- pretrained checkpoints
- configuration files
- sample microscope images
- export scripts for digital-twin reconstruction

Recommended minimal structure:

```text
code/vision/
├── Pose_and_Depth.ipynb
├── checkpoints/
├── configs/
├── samples/
└── README.md
```

The repository is deliberately honest here: the current public release prepares the slot and the tutorial path, but it does not claim to include unreleased perception code.
