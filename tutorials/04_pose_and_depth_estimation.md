# Pose and Depth Estimation Tutorial

This tutorial explains how to run the perception notebook used in this repository.

## File

- `code/vision/Pose_and_Depth.ipynb`

## Recommended workflow

1. Open the notebook in Jupyter Notebook or VS Code.
2. Update the dataset path variables in the notebook.
3. Check the label file and image-folder paths.
4. Run the data loading and preprocessing cells.
5. Run the dataset split cells.
6. Run the model training cells.
7. Run the evaluation cells to obtain pose/depth prediction results.
8. Save the model weights and outputs if needed.

## Notes

- Please update all local file paths before running the notebook.
- If you use your own dataset, keep the label format consistent with the notebook.
- The notebook can be used as the perception side of deployment-oriented digital-twin reconstruction.
